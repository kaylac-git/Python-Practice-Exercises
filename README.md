# Python Practice Exercises

A collection of practice exercises completed as part of CS50's Introduction to Programming with Python at Harvard X, 2025–2026.

## Topics
- Variables and data types
- Loops and conditionals
- Functions
- Lists and dictionaries
- File handling

## Structure
Each folder corresponds to a unit. Files are named descriptively (e.g. `fibonacci.py`, `bubble_sort.py`).

## Running
Requires Python 3.x. Run any file with:
    python filename.py
